what you need
system
Install Node.js LTS. If you already have Node and npm, you are ready for Artemis.
check first
Run node --version and npm --version. Version numbers mean you can skip the system install step.
use this window
macOS: Terminal. Windows: PowerShell.
macOS install

The simplest path for a Mac user who wants the CLI working without extra setup.

step 01

open terminal

Go to Applications → Utilities → Terminal. Keep the same window open while you install.

step 02

check node and npm

If both commands return a version number, skip straight to install Artemis.

macos terminal
node --version
npm --version
step 03

install node if missing

The easiest route is the Node.js LTS installer from nodejs.org. Or via Homebrew:

macos terminal
brew install node
step 04

install artemis

macos terminal
npm install -g @rtemis
step 05

verify and open

macos terminal
artemis --version
artemis
windows install

For a normal Windows machine using PowerShell, not an advanced terminal setup.

step 01

open powershell

step 02

check node and npm

If both commands answer with versions, Node is already installed.

windows powershell
node --version
npm --version
step 03

install node if missing

Use the Node.js LTS installer from nodejs.org. Or via winget:

windows powershell
winget install OpenJS.NodeJS.LTS
step 04

install artemis

windows powershell
npm install -g @rtemis
step 05

verify and open

windows powershell
artemis --version
artemis
first launch
01

boot the cli

Open one terminal window, confirm the version, let the first interactive run start.

terminal
artemis --version
artemis
02

set the runtime

Enter API URL, model, API key, provider alias, and docs fallback first. Leave Telegram and doublekill for later.

03

run a health pass

Check health, providers, and sessions before pointing Artemis at a real repo.

terminal
artemis doctor
artemis providers
artemis sessions
04

start with a small task

Start with one short prompt. Add memory with artemis artemis-md and checkpoints with wordup.

terminal
artemis run "make this repo easier to understand"
artemis niko "plan the next improvement"
full uninstall

Pick the platform wipe that matches your machine, then confirm the command is gone.

Remove the global package first, then confirm artemis --version fails (macOS) or Get-Command artemis returns nothing (Windows).

terminal
npm uninstall -g @rtemis
macOS

full wipe — default macOS locations

Run from each project root where you used Artemis.

macos terminal
npm uninstall -g @rtemis
npm cache clean --force
rm -rf "$(npm config get cache)/_npx"

rm -rf "$HOME/.artemis" \
"$HOME/.config/artemis" \
"$HOME/.cache/artemis" \
"$HOME/.local/share/artemis" \
"$HOME/.local/state/artemis" \
"$HOME/Library/Application Support/Artemis" \
"$HOME/Library/Caches/Artemis" \
"$HOME/Library/Logs/Artemis"

find . -type f \( -name 'Artemis.MD' -o -name 'WordUP*.md' \) -delete
hash -r
windows

full wipe — default windows locations

Run from each project root where you used Artemis.

windows powershell
npm uninstall -g @rtemis
npm cache clean --force
$npmCache = npm config get cache
Remove-Item -Path (Join-Path $npmCache "_npx") -Recurse -Force -ErrorAction SilentlyContinue

@(
"$env:USERPROFILE\.artemis"
"$env:APPDATA\artemis"
"$env:LOCALAPPDATA\artemis"
"$env:APPDATA\Artemis"
"$env:LOCALAPPDATA\Artemis"
) | ForEach-Object {
Remove-Item -Path $_ -Recurse -Force -ErrorAction SilentlyContinue
}

Get-ChildItem -Path . -Recurse -Force -Include "Artemis.MD","WordUP*.md" |
Remove-Item -Force -ErrorAction SilentlyContinue
